# PINTOUR

Pintour adalah website yang menyediakan layanan paket wisata.

Website ini menyediakan fitur :

- Login & Registrasi
- Email Auth
- Display & Checkout Wisata
- Admin Panel

## Dibuat dengan menggunakan

Website ini dibuat dengan menggunakan :

- Html, CSS & Bootstrap
- PHP (Laravel Framework)
- MySQL
